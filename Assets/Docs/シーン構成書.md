## TitleScene

### Hierarchy

```
TitleScene
├── EventSystem
├── TitleCanvas
│   ├── Background
│   ├── TitleText              ← 英語のみ（タイトルロゴ）
│   ├── SubTitle               ← 英語のみ
│   ├── MainMenuPanel
│   │   ├── AIBtn / OnlineBtn / RulesBtn
│   │       └── Label          ← 日本語（AIと対戦/オンライン（準備中）/ルール）
│   ├── AILevelPanel          ※起動時 activeSelf=false
│   │   ├── AITitle            ← 「難易度を選択」
│   │   ├── EasyBtn / NormalBtn / HardBtn / BackBtn
│   │       └── Label          ← 日本語（かんたん/ふつう/むずかしい/← もどる）
│   └── TitleController
├── AIController
├── MenuBtn
├── MenuBackdrop              ← メニュー開時の半透明黒オーバーレイ（非表示）
├── MenuDropdown              ← ドロップダウン（非表示）
│   ├── RulesBtn              ← 「ルール確認」
│   └── TitleBtn              ← 「タイトルへ」
├── RulesPanel                ← ルール説明オーバーレイ（非表示）
│   ├── ContentPanel（ScrollRect）
│   │   └── RulesText         ← 日本語ルール説明（NotoSansJP）
│   └── CloseBtn              ← 「閉じる」
└── Main Camera
```

### コンポーネント一覧

> 全テキスト（TitleText/SubTitle除く）：NotoSansJP-Regular SDF（Dynamicモード）

| GameObject | コンポーネント | 備考 |
|---|---|---|
| EventSystem | EventSystem, InputSystemUIInputModule | StandaloneInputModuleは使わない |
| TitleCanvas | Canvas, CanvasScaler, GraphicRaycaster | |
| TitleText / SubTitle | TextMeshProUGUI | 英語テキストのみ（タイトルロゴ） |
| AIBtn / OnlineBtn / RulesBtn | Image, Button | |
| AILevelPanel | Image | 起動時非表示。上下余白均等 |
| TitleController | TitleScreenUI | GameSettings設定後にBattleSceneをロード |
| RulesPanel | Image | 起動時非表示。フルスクリーンオーバーレイ |
| Main Camera | Camera, AudioListener, UniversalAdditionalCameraData | |

## BattleScene

### Hierarchy

```
BattleScene
├── EventSystem
├── GameManager
├── GameCanvas
│   ├── Background
│   ├── PhaseLabel
│   │   └── Txt                ← 「Turn 1 | 獲得フェーズ」形式
│   ├── Player1Panel           ← 幅280px・左端アンカー
│   │   ├── Highlight          ※フェーズボーダー（4本帯）※点滅はしない
│   │   ├── TurnLabel          ※起動時 activeSelf=false
│   │   │   └── Txt            ← 「YOUR TURN」
│   │   ├── PlayerName / Txt
│   │   ├── HPLabel / Txt      ← 「HP:15/15」形式
│   │   ├── ShieldText         ← 「シールド 0」形式・常時表示
│   │   ├── HPBar              ※activeSelf=false（非表示）
│   │   ├── SlotsLabel         ※activeSelf=false（非表示）
│   │   ├── SlotsContainer     ← 幅258px・中央アンカー
│   │   │   ├── Row1〜Row5     ← 幅248px
│   │   │   │   ├── Slot0〜SlotN
│   │   │   │   │   ├── Txt
│   │   │   │   │   └── LockedOverlay  ※ロック時に表示
│   │   │   │   │       └── Txt        ← 「X」または「X2」等
│   │   │   │   └── RowRomanLabel      ← 「Ⅰ〜Ⅴ」・左端外側配置
│   │   ├── ResLabel / Txt     ← 「リソース (0/6)」
│   │   └── ResContainer       ※リソースチップを動的生成
│   ├── Player2Panel           ※Player1Panelと同構造
│   │   └── （RowRomanLabelは右端外側配置）
│   ├── PoolPanel              ← 幅560px・高さ240px
│   │   ├── Highlight          ※フェーズボーダー（Pulse点滅）
│   │   ├── Title              ※activeSelf=false（非表示）
│   │   └── PoolRow0〜2        ← 高さ44px
│   │       ├── PoolLabel      ← 「プール 0:」
│   │       ├── ChipsContainer
│   │       └── TakeButton     ← 高さ36px（2段レイアウト）
│   │           ├── Label      ← 「取得 x2」（上半分）
│   │           ├── PreviewChip0  ← リソース色ドット（下半分・左）
│   │           └── PreviewChip1  ← リソース色ドット（下半分・右）
│   ├── ControlPanel           ← 幅600px
│   │   ├── Highlight          ※フェーズボーダー（Pulse点滅）
│   │   ├── SelectedLabel      ← 「選択中: -」
│   │   ├── AttackBtn / DefenseBtn / DisruptBtn
│   │   ├── RowBtn1〜5         ← ラベル「Ⅰ〜Ⅴ」
│   │   ├── EndAssignBtn       ← 「配置完了」
│   │   └── ResetBtn           ← 「リセット」
│   ├── GameOverPanel          ※起動時 activeSelf=false
│   │   ├── GameOverText / ToTitleBtn / RestartBtn
│   └── TutorialHintPanel      ← Y=260
├── AIController
├── MenuBtn / MenuBackdrop / MenuDropdown
├── RulesPanel
├── AnnouncementOverlay        ← CanvasGroup・最前面配置
│   ├── TextBackground         ← 半透明黒下地（α=0.55・全幅×200px）
│   ├── MainText               ← GAME START / ACQUIRE / ASSIGN / RESOLVE
│   └── SubText                ← 「あなたのターン」/「AIのターン」（NotoSansJP）
└── Main Camera
```

| SlotsContainer/RowN/SlotM | Image, Button | M=0〜N-1 |
| SlotM/LockedOverlay | Image | ロック時に表示・赤色 |
| HPLabel/Txt | TextMeshProUGUI | 「HP:15/15」形式 |
| ShieldText | TextMeshProUGUI | 「シールド 0」形式・常時表示・NotoSansJP |

| AnnouncementOverlay/SubText | TextMeshProUGUI | NotoSansJP。ターン所有者表示 |
| Main Camera | Camera | |
