## シーン一覧

| シーン名 | 役割 | Build Index | パス |
|---|---|---|---|
| TitleScene | タイトル・モード選択画面 | 0 | Assets/Scenes/TitleScene.unity |

> ⚠️ SampleSceneは使用しない。

## TitleScene

### Hierarchy

```
TitleScene
├── EventSystem
├── TitleCanvas
│   ├── Background
│   ├── TitleText
│   ├── SubTitle
│   ├── MainMenuPanel
│   │   ├── LocalBtn / AIBtn / OnlineBtn
│   │       └── Label
│   ├── AILevelPanel          ※起動時 activeSelf=false
│   │   ├── AITitle
│   │   ├── EasyBtn / NormalBtn / HardBtn / BackBtn
│   │       └── Label
│   └── TitleController
├── AIController
└── Main Camera
```

### コンポーネント一覧

| GameObject | コンポーネント | 備考 |
|---|---|---|
| EventSystem | EventSystem, InputSystemUIInputModule | StandaloneInputModuleは使わない |
| TitleCanvas | Canvas, CanvasScaler, GraphicRaycaster | |
| TitleText / SubTitle | TextMeshProUGUI | 英語テキストのみ（日本語文字化けのため） |
| LocalBtn / AIBtn / OnlineBtn | Image, Button | |
| TitleController | TitleScreenUI | GameSettings設定後にBattleSceneをロード |
| Main Camera | Camera, AudioListener, UniversalAdditionalCameraData | |

## BattleScene

### Hierarchy

```
BattleScene
├── EventSystem
├── GameManager
├── GameCanvas
│   ├── Background
│   ├── PhaseLabel
│   │   └── Txt
│   ├── Player1Panel
│   │   ├── Highlight              ※フェーズボーダー（4本帯: Top/Bottom/Left/Right）
│   │   ├── TurnLabel              ※起動時 activeSelf=false
│   │   │   └── Txt
│   │   ├── PlayerName
│   │   │   └── Txt
│   │   ├── HPLabel
│   │   │   └── Txt                ← HP表示（HP:20/20形式）
│   │   ├── ShieldText             ← Shield値常時表示（Shield 0形式）
│   │   ├── HPBar                  ※activeSelf=false（非表示）
│   │   ├── SlotsLabel
│   │   │   └── Txt
│   │   ├── SlotsContainer
│   │   │   ├── Row1〜Row5
│   │   │   │   └── Slot0〜SlotN
│   │   │   │       ├── Txt
│   │   │   │       └── LockedOverlay  ※ロック時に表示
│   │   │   │           └── Txt        ← "X" テキスト（赤・太字）
│   │   ├── ResLabel
│   │   │   └── Txt
│   │   └── ResContainer           ※リソースチップを動的生成
│   ├── Player2Panel               ※Player1Panelと同構造
│   ├── PoolPanel
│   │   ├── Highlight              ※フェーズボーダー（4本帯）
│   │   ├── Title / PoolRow0〜2
│   ├── ControlPanel
│   │   ├── Highlight              ※フェーズボーダー（4本帯）
│   │   ├── SelectedLabel / AttackBtn〜DisruptBtn / RowBtn1〜5 / EndAssignBtn / ResetBtn
│   └── GameOverPanel              ※起動時 activeSelf=false
│       ├── GameOverText / ToTitleBtn / RestartBtn
├── AIController
└── Main Camera
```

### コンポーネント一覧

| GameObject | コンポーネント | 備考 |
|---|---|---|
| EventSystem | EventSystem, InputSystemUIInputModule | StandaloneInputModuleは使わない |
| GameManager | GameManager, GameUI | GameUIはenabled=false（レガシー） |
| GameCanvas | Canvas, CanvasScaler, GraphicRaycaster, GameUINew, EffectManager | EffectManager追加 |
| Background | Image | |
| PhaseLabel/Txt | TextMeshProUGUI | Resolve中は発動内容を表示 |
| Player1Panel / Player2Panel | Image, PlayerPanelUI | |
| Player1Panel/Highlight | RectTransform + 4子(Top/Bottom/Left/Right Image) | フェーズボーダー・黄色 |
| HPLabel/Txt | TextMeshProUGUI | "HP:20/20"形式 |
| ShieldText | TextMeshProUGUI | "Shield 0"形式・常時表示 |
| HPBar | Slider | activeSelf=false（非表示） |
| SlotsContainer/RowN | Image | N=1〜5 |
| SlotsContainer/RowN/SlotM | Image, Button | M=0〜N-1 |
| SlotsContainer/RowN/SlotM/LockedOverlay | Image | ロック時に表示・赤色 |
| SlotsContainer/RowN/SlotM/LockedOverlay/Txt | TextMeshProUGUI | "X"テキスト・赤・太字 |
| ResContainer | （空）| リソースチップをGameUINewが動的生成 |
| PoolPanel | Image, PoolPanelUI | |
| PoolPanel/Highlight | RectTransform + 4子Image | フェーズボーダー |
| ControlPanel | Image, ControlPanelUI | |
| ControlPanel/Highlight | RectTransform + 4子Image | フェーズボーダー |
| GameOverPanel | Image | 起動時非表示 |
| GameOverPanel/ToTitleBtn | Image, Button | TitleSceneへ遷移。グレー系色 |
| GameOverPanel/RestartBtn | Image, Button | ゲームを再スタート |
| AIController | AIController | GameSettings.Mode==AI 以外は自動無効化 |
| Main Camera | Camera | |
