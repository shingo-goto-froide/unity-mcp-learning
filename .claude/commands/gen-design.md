# 設計フェーズ：設計書生成

以下のファイルをすべて読み込んでください：
- `Assets/Docs/PROJECT.md`（プロジェクト固有の制約・注意事項を把握する）
- `Assets/Docs/仕様書_*.md`（最新版）
- `Assets/Docs/設計書_*.md`（既存があれば読み込む）

読み込んだ後、CLAUDE.md の設計書テンプレートに従い `Assets/Docs/設計書_v{バージョン}.md` を生成してください。

**生成する内容：**
1. スクリプト一覧・依存関係ツリー
2. Enum・ScriptableObject・静的クラス定義
3. 各クラスの変数・イベント・メソッド定義（pure C# / MonoBehaviour / Singleton を明記）
4. フォルダ構成

> ⚠️ フォルダ構成・クラス名・使用ライブラリ等のプロジェクト固有の制約は PROJECT.md の注意事項を参照すること。

完成したら「設計書が完成しました。人の確認後に Git commit してください」と通知する。
