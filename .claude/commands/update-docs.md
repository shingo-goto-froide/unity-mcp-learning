# ドキュメント更新

以下のファイルをすべて読み込んでください：
- `Assets/Docs/仕様書_*.md`（最新版）
- `Assets/Docs/設計書_*.md`（最新版）
- `Assets/Docs/シーン構成書.md`
- `Assets/Scripts/` 配下の全スクリプト

読み込んだ後、実際の実装に合わせてドキュメントを更新してください。

**更新ルール（CLAUDE.md 準拠）：**

| 更新すべきケース | 対象ドキュメント |
|---|---|
| ゲームルール・フェーズ順序・効果量の変更 | 仕様書 |
| 新クラス・メソッド・変数の追加・変更・削除 | 設計書 |
| Hierarchy・コンポーネント・Inspector値の変更 | シーン構成書 |
| バグ修正のみ（インターフェース変化なし） | 更新不要 |
| コメント追記・数値チューニングのみ | 更新不要 |

**更新後の確認：**
- PROJECT.md の変更履歴テーブルに今日の日付・対象・内容を追記する
- 更新完了後「ドキュメントを更新しました。Git commit してください」と通知する
