# 企画フェーズ：仕様書作成

以下のファイルをすべて読み込んでください：
- `Assets/Docs/PROJECT.md`（プロジェクト固有情報・注意事項を把握する）
- `Assets/Docs/仕様書_*.md`（既存の仕様書があれば読み込む）

読み込んだ後、CLAUDE.md の仕様書テンプレートに従い、仕様書を対話形式で作成してください。

1. PROJECT.md のプロジェクト概要・注意事項を踏まえて仕様を詰める
2. 仕様書が固まったら `Assets/Docs/仕様書_v{バージョン}.md` として保存する
3. 完成したら「仕様書が完成しました。Git commitのタイミングです」と通知する

> ⚠️ プロジェクト固有の制約（フォント・アセットパス・禁止事項等）は PROJECT.md の注意事項を参照すること。
