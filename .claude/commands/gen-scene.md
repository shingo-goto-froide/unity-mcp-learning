# 実装フェーズ：シーン構築

以下のファイルをすべて読み込んでください：
- `Assets/Docs/PROJECT.md`（プロジェクト固有の制約・注意事項を把握する）
- `Assets/Docs/シーン構成書.md`
- `Assets/Docs/設計書_*.md`（最新版）

読み込んだ後、シーン構成書に従いシーンを構築してください。

**構築手順：**
1. シーン構成書のシーン一覧に記載されたシーンを対象にする
2. Hierarchy に従い GameObject を作成
3. コンポーネント一覧に従いコンポーネントをアタッチ
4. Inspector 参照設定に従いフィールドを接続
5. Canvas / CanvasScaler 設定を行う

> ⚠️ シーン名・フォント・禁止コンポーネント・レイアウト詳細等のプロジェクト固有の制約は PROJECT.md の注意事項を参照すること。

構築完了後：「シーン構築が完了しました。人の確認後に Git commit してください」と通知する。
