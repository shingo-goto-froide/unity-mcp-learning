# テストフェーズ：実装と仕様書の差分確認

以下のファイルをすべて読み込んでください：
- `Assets/Docs/仕様書_*.md`（最新版）
- `Assets/Docs/設計書_*.md`（最新版）
- `Assets/Scripts/` 配下の全スクリプト

読み込んだ後、以下の観点で差分を報告してください：

**確認項目：**
1. **仕様書との差異** - 実装されているゲームルール・フェーズ処理が仕様書と一致しているか
2. **設計書との差異** - クラス・メソッド・変数が設計書と一致しているか（追加・変更・削除）
3. **未実装項目** - 仕様書・設計書に記載されているが実装されていない機能
4. **ドキュメント未反映** - 実装されているが仕様書・設計書に記載されていない機能

**報告フォーマット：**
```
## 差分レポート

### 仕様書との差異
- （差異があれば列挙）

### 設計書との差異
- （差異があれば列挙）

### 未実装項目
- （あれば列挙）

### ドキュメント未反映の実装
- （あれば列挙）

### 推奨アクション
- （軽微バグ → CLI即修正 / 仕様・設計変更 → 設計フェーズへ戻る）
```
